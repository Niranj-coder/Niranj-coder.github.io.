Kali-Termux-Pro

Automated Kali NetHunter (rootless) setup in Termux – Work in progress.

Description

Simple automation script to get Kali NetHunter running in Termux without manual hassle. Built because manual PRoot setup gets annoying after switching ROMs or resetting your phone.

Note: Still under active development — expect possible bugs. Report any device-specific issues you encounter.

What It Does

  • Updates Termux repositories (standard pkg update/upgrade)
  • Installs required dependencies (proot, wget, curl, git)
  • Fetches the latest official NetHunter rootless install script
  • Sets execute permissions and runs the installer automatically

Quick Install

pkg update -y && pkg upgrade -y && pkg install git -y && git clone https://github.com/Niranj-coder/kali-termux-pro && cd kali-termux-pro && chmod +x install.sh && ./install.sh

Requirements

  • Termux app installed on an Android device
  • Internet connection for package downloads and script fetch
  • No root required (uses PRoot for rootless NetHunter)

Main Script & Repo

  • Main executable: install.sh
  • Full repository: Open on GitHub →

Troubleshooting

If something fails (e.g., dependency issues, script download error, or device-specific problems), open an issue on GitHub with details about your Android version, Termux version, and any error messages shown.
